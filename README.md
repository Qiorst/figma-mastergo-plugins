# MasterGo Plugins

This repository stores multiple MasterGo plugins in separate folders.

## Plugins

### Image Tuner

Path: `plugins/image-tuner`

Features:

- Preview selected image layers.
- Adjust brightness, contrast, saturation, and hue.
- Apply tuned images back to MasterGo layers.
- Preserve original image source for reset.
- Reset parameters and restore the original image.
- Detect external image source changes and reset parameters automatically.

## Version Tags

Each plugin uses its own tag prefix:

- `image-tuner-v1.0.0`

Future plugins should use tags like:

- `plugin-name-v1.0.0`
